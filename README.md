به پروژه تشخیص گیاهان خانگی خیلی خوش اومدین! 

زبان اصلی پایتون و کتابخونه اصلی برای آموزش مدل pytorch است. 
ددیتای مورد استفاده در این پروژه دیتای house_plant_species از سایت kaggle است. 

توی این ریپوزیتوری برای شما تمامی فایل های پروژه تشخیص گیاهان خانگی رو قرار دادم. توی فایل house_plant_species_detector.ipynb کد های آموزش این مدل قرار گرفته. همچنین تصاویر گراف های ارزیابی این مدل هم اینجا قرار دارن. شما میتونید در اینت تصاویر conf-matrix و accuracy-loss_-plot رو مشاهده کنید.
امیدوارم از این پروژه خوشتون بیاد و براتون مفید باشه:)
